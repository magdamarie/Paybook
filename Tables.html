Tables

How to make tables.

Loot at docs on MDN.


  • Use to structure and represent data in your website.
  • Inside table can create Table Row and Table Data.
    • Table Row 'tr'
    • Table Data/Cell 'td'
  • Various Atributes available for table to change appearance.
  • Most Attributes are Deprecated (Thumbs Down) - not recommended - resist when possible
    • Lines
    • Cellspacing
      • This attribute defines the sizes of the space between two cells in a percentage value or pixels.
      • The atribute is applied both horizontally and vertically, to the space between:
        • the top of the table and the cells of the first row;
        • the left of the table and the first column:
        • the right of the table and the last column; and
        • the bottom of the table and the last row.
    • HTML is for structure, not style.
